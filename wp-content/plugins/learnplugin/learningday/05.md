# link one to another page

example:

    - add-new.php
    - view-all.php

method to add pages:

include : include a file multiple times. If the file is not found, PHP throws a warning but the script continues running.
example: include('file.php'); include 'file.php';

include_once : include a file only once. If the file is not found, PHP throws a warning but the script continues running.

require : include a file multiple times. If the file is not found, PHP throws a fatal error and the script stops.
example: require('file.php'); require 'file.php';

require_once : include a file only once. If the file is not found, PHP throws a fatal error and the script stops.
