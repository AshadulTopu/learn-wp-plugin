# Post Form Data with Form Validation

there has several ways to validate form data in wordpress.

1. wp_validate_nonce()
2. wp_verify_nonce()
3. wp_check_referer()
4. wp_nonce_field()

## wp_validate_nonce()

this function is used to validate nonce values.

```php
wp_validate_nonce( $nonce, $action );
```

## wp_verify_nonce()

this function is used to verify nonce values.

```php
wp_verify_nonce( $nonce, $action );
```

## wp_check_referer()

this function is used to check referer values.

```php
wp_check_referer( $action );
```

## wp_nonce_field()

this function is used to generate nonce fields.

```php
wp_nonce_field( $action, $name );
```
