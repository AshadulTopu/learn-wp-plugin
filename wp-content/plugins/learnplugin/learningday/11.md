# Delete page when plugin deactivate / uninstall

## wp_insert_post()

when create page using wp_insert_post() function store post id, then search post id on wp_options table on database.

## add_option()

```php
<?php
add_option( string $option, mixed $value, string $deprecated, bool|null $autoload = null ): bool
```

reference: https://developer.wordpress.org/reference/functions/add_option/

delete page when plugin deactivate/uninstall
step 1: get page id from wp_options table on database
step 2: delete page using wp_delete_post()

## wp_delete_post() function

```php
<?php
wp_delete_post( int $post_id, bool $force_delete = false ): WP_Post|false|null
```
